1.5.7i deutsch ist ein Service Pack für 1.5.7h deutsch
Damit wird PHP 8.3.x vollständig unterstützt.
Die integrierten Module und 3rd Party Komponenten werden auf den Stand von August 2024 gebracht.
1.5.7i behebt auch alle Fehler die seit Erscheinen von 1.5.7h auf Github gemeldet wurden.

Aktualisierte Komponenten gegenüber 1.5.7h:
- Braintree Zahlungsart
- CKEditor

1.5.7i bringt als Beispiel für ein via Plugin Manager installierbares Modul das Modul Sofortsuche (Instant Search) in der Version 4.0.3 mit.
Infos zu diesem Modul hier:
https://www.zen-cart-pro.at/knowledgebase/sofortsuche-instant-search-fuer-zen-cart-1-5-7-deutsch/
Sie können es unter Module > Plugin Manager installieren und aktivieren, wenn Sie es in Ihrem Shop nutzen wollen.
Falls Sie dieses Modul bereits in Ihrem 1.5.7h Shop nutzen, dann müssen Sie die folgenden neuen Dateien oder Ordner nicht hochladen, da sie bereits vorhanden sind.
