1.5.7i deutsch ist ein sehr umfangreicher Service Pack für 1.5.7 deutsch
Es integriert die fast vollständige Codebasis der aktuellen amerikanischen Version 1.5.8 und ein Großteil der amerikanischen Version 2.0.0 in die deutsche Zen Cart Version 1.5.7.
Damit wird PHP 8.3.x vollständig unterstützt.
Die integrierten Module und 3rd Party Komponenten werden auf den Stand von August 2024 gebracht.
Der Adminbereich wird mit dieser Version auf ein modernes, einheitliche Layout gebracht.
Das bislang in der Grundinstallation enthaltene Zahlungsmodul Sofortüberweisung (sofort_su) wurde entfernt, da es veraltet ist und von dem neuen Sofort Eigentümer Klarna nicht mehr unterstützt wird.

Die Umstellung der Sprachdateien, die die amerikanische Version 1.5.8/2.0.0/2.1.0 bereits mitbringt, wird in einer für 2025 geplanten deutschen Zen Cart Version 2.1.0 mit vollständiger PHP 8.4 Unterstützung nachgezogen.
Dadurch kann in 1.5.7i weiterhin mit den gewohnten Sprachdefinitionen gearbeitet werden. 

1.5.7i behebt auch alle Fehler die seit Erscheinen von 1.5.7h auf Github gemeldet wurden.

Aktualisierte Komponenten:
- PHP Mailer Update auf 6.9.1
- MobileDetect Update auf 3.74.3
- CKEditor Update auf 4.22.1
- TinyMCE Editor Update auf 7.0.0
- braintree_web Update auf 3.106.0, braintree_php Update auf 6.19.0 
- FontAwesome Update auf 6.4.0
- jQuery Update auf 3.7.1
- Image Handler Update auf 5.3.4 mit webp Unterstützung

Neue Funktionalitäten:
- Kunden können Kundengruppen zugeordnet werden. Plugins könnten bestimmte Funktionen nur für bestimmte Kundengruppen bereitstellen.
- Unterstützung des Plugin Managers für Module, die bereits dieses Format nutzen
- Beim Löschen eines Kunden kann entweder der gesamte Kundeneintrag gelöscht werden (Standard) oder mit "Nur vergessen" nur die persönlichen Daten anonymisiert werden.
- Logfilemanager integriert (automatisches Löschen von Logfiles nach einem bestimmten Zeitraum)
- In der Shopadministration können unter Konfiguration > Admin Layout bis zu 10 verschiedene externe Links für die tägliche Arbeit definiert werden, die dann im Adminmenü erscheinen.
- Gedacht ist das für Links, die man ständig im Shopalltag braucht, z.B. Links zu den Administrationsoberflächen Ihrer Payment Provider oder Versanddienstleister.

Verbesserungen:
- Vollständige Unterstützung von PHP 8.3.x
- Neustrukturierung wichtiger Funktionsdateien
- Performanceoptimierung durch Zentralisierung von Klassen und Funktionen 
- Admin Layout durchgehend augenfreundlich und mobilefreundlich
- Suchfunktionen im Adminbereich verbessert
- mehrfache Sprachdefinitionen wurden entfernt
- Artikelmerkmale stehen nun auch in den anderen Artikeltypen zur Verfügung
- Steuersatz der Versandkosten wird nun in der Datenbank bei der Bestellung gespeichert
- Performanceverbesserungen für Shops mit großen Datenbanken
- Neue Zusatzinformationen auf der Adminseite Bestellungen
- Updatecheck zur englischen Version im Header entfernt, der Updatecheck befindet sich nun unter Tools > Serverinfo und auf der Hilfeseite
- back to top Pfeil bei Scrollen im Adminbereich

Wer bereits 1.5.7h verwendet kann mit dem speziellen Updatepaket in UPDATE-VON-157h-AUF-157i sehr leicht auf 1.5.7i aktualisieren.
Anleitung dazu in der liesmich.txt im Ordner UPDATE-VON-157h-AUF-157i

Falls Sie bereits Version 1.5.7 deutsch verwenden, aber noch nie auf 1.5.7h aktualisiert haben, dann verwenden Sie das spezielle Updatepaket UPDATE-VON-157-AUF-157i.
Anleitung dazu in der liesmich.txt im Ordner UPDATE-VON-157-AUF-157i

Für das Update einer älteren deutschen Zen Cart Version als 1.5.7 (Sie verwenden z.B. derzeit 1.5.6 oder 1.5.5), führen Sie bitte ein normales Update mit dem kompletten 1.5.7i Paket durch!
Detaillierte Updateanleitung für das Update solch älterer Versionen:
* https://updateanleitung.zen-cart-pro.at
